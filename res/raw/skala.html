Skala

  • Rumus

    Skala = jarak pada peta / jarak sebenarnya
  • Jarak pada peta = Skala x jarak sebenarnya
  • Jarak sebenarnya = jarak pada peta / sakala

  • Contoh

    Panjang halaman rumah Pak Amir 600 cm. Lebarnya 400 cm. Berapakah panjang dan lebar rumah Pak Amir pada peta jika diketahui Skala = 1:100.
    Jawab :
    Jarak pada peta = Skala X jarak sebenarnya
    Panjang pada peta = 1 / 100 X 600
    = 600 / 100
    = 6 cm.
    Lebar pada peta = 1/100 X 400
    = 400 / 100
    = 4 cm.

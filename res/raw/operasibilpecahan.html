Operasi Bilangan Pecahan

Penjumlahan

Rumus :
Penyebut sama nilainya:
a/n + b/n = (a + b) / n
Penyebut tidak sama nilainya:
a/n + b/m = ((a x m)+(bxn)) / (n x m)

Pengurangan

Rumus :
Penyebut sama nilainya:
a/n - b/n = (a - b) / n
Penyebut tidak sama nilainya:
a/n - b/m = ((a x m)-(bxn)) / (n x m)

Perkalian

Rumus :
a/b x c/d = (a x c) / (b x d)

Pembagian

Rumus :
a/b : c/d = (a x d) / (b x c)
